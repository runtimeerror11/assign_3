## 📋 Overview

This project classifies cotton plant images into **4 categories**:

| Class | Description |
|-------|-------------|
| `diseased cotton leaf` | Leaf showing disease symptoms |
| `diseased cotton plant` | Whole plant showing disease |
| `fresh cotton leaf` | Healthy leaf |
| `fresh cotton plant` | Healthy whole plant |

The model achieves **100% test accuracy** on the [Cotton Disease Dataset](https://www.kaggle.com/datasets/janmejaybhoi/cotton-disease-dataset).

```
├── requirements.txt                 # Dependencies
├── README.md                        # This file
└── app/
    ├── __init__.py
    ├── config.py                    # ⚙️ THE SINGLE FILE to change for model swap / ensemble
    ├── routes.py                    # Flask routes (model-agnostic)
    └── templates/
        └── index.html               # Web UI
```

### 🔑 Key Design Decision

**All model logic lives in `app/config.py`.**  
If you want to:
- **Swap to a different model** → change `load_model()`
- **Add ensemble of multiple models** → return an `nn.Module` whose `forward()` averages logits
- **Change transforms** → edit `EVAL_TRANSFORM`
- **Add / rename classes** → edit `CLASS_NAMES`

**No other file needs to be touched.** Routes and UI are completely model-agnostic.

## 🚀 Quick Start

### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

> **Note:** If you have a CUDA-capable GPU, install the GPU version of PyTorch first:
> ```bash
> pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
> ```

### 2. Verify Model Weights

Make sure the `models/` folder contains `efficientnet_b3_cotton.pth` (and optionally `resnet50_cotton.pth`).  
These files are generated by running the training notebook `smai-model1.ipynb`.

### 3. Run the App

```bash
python run.py
```

Open your browser at **http://localhost:5000** 🎉

## 🔄 Changing the Model (or Adding Ensemble)

Open `app/config.py` and modify **only** the `load_model()` function:

```python
def load_model() -> nn.Module:
    # Example: switch to ResNet50
    model = models.resnet50(weights=None)
    model.fc = nn.Linear(model.fc.in_features, NUM_CLASSES)
    state = torch.load(os.path.join(MODEL_DIR, "resnet50_cotton.pth"), map_location=DEVICE)
    model.load_state_dict(state)
    model.to(DEVICE)
    model.eval()
    return model
```

For **ensemble**, create a wrapper:

```python
class EnsembleModel(nn.Module):
    def __init__(self, models):
        super().__init__()
        self.models = nn.ModuleList(models)

    def forward(self, x):
        logits = [m(x) for m in self.models]
        return torch.stack(logits).mean(dim=0)
```
